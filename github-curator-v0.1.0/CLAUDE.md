# CLAUDE.md

This file is functionally a symlink to `AGENTS.md`. Read that file. It
contains the project overview, build commands, code style, testing,
security, PR conventions, and provenance.

GitHub does not preserve symlinks reliably across forks and template
clones, so this 3-line note is a portable substitute. The Agent Skills
specification, AGENTS.md standard, and the Anthropic memory hierarchy
all permit this pattern. The Vercel and Next.js repos use the same
approach: see https://github.com/vercel/next.js/blob/canary/AGENTS.md
which states "CLAUDE.md is a symlink to AGENTS.md. They are the same
file."
